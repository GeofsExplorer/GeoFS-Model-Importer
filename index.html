Import Any 3D Model

GeoFS Model Importer is a powerful UserScript that allows you to easily load and position GLTF (.gltf) and GLB (.glb) models directly into the GeoFS world, giving you complete freedom for custom scenery or aircraft visualization.

Features

Static Scenery Placement

Place any model (buildings, vehicles, landmarks) as a **static object** exactly where your aircraft is located, matching the ground elevation and heading.

Aircraft Model Replacement

Replace the visual model of your active aircraft. The imported model will dynamically track your plane's position, heading, pitch, and roll.

Dynamic Scale & CoM Marker

Adjust the model's scale on the fly using the dedicated slider, and optionally visualize the Center of Mass (CoM) with a toggleable marker.

Installation Guide

GeoFS Model Importer is a UserScript, requiring a browser extension manager like Tampermonkey or Violentmonkey to function.

Step 1: Get a Script Manager

Install one of the following browser extensions: Tampermonkey (Recommended) or Violentmonkey.

Step 2: Install the Script

Click the button below to install the script directly into your manager:

Install GeoFS Model Importer Script

How to Use

  1. Load GeoFS and open the floating "Model Importer" control panel at the bottom left.
  2. Set your desired Scale using the input field or slider.
  3. Click "Select Model" to choose your .gltf or .glb file.
  4. Click "Place Here" for a static object or "Use as Aircraft" to replace your plane's visual model.
